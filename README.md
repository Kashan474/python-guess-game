# 🎯 Number Guessing Game — Python

A simple command-line Number Guessing Game built in Python.
The player sets a custom range, and the program generates a
secret number. The goal is to guess it in as few attempts as possible!

---

## 🚀 Features

- Custom Min & Max range input by the user
- Randomly generated secret number using Python's `random` module
- Attempt counter to track how many guesses were made
- Input validation with `try/except` for error-free gameplay
- Real-time feedback: too high, too low, or correct!

---

## 🛠️ Concepts Used

- `random.randint()` for random number generation
- Python type hints (`int` annotations)
- `while` loop for continuous gameplay
- Conditional statements (`if / elif / else`)
- Exception handling (`try / except`)
- User input & formatted output (`f-strings`)

---

## ▶️ How to Run

```bash
python guess_game.py
```

Then follow the prompts:
1. Enter your minimum number
2. Enter your maximum number
3. Start guessing!
